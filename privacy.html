Privacy Policy

User Data Collection

Users game statistics (high scores and games played) are collected by Arcade Classics to a public database.

This database is used to create a leaderboard system in app.

Users can opt in to sending error reports to Arcade Classics.

Data handling and storage

Upon opening the extension the user is assigned a random id as well as a random username (that they can change later).

This id and username will be used to identify users when game statistics (high scores and games played) are automatically uploaded to the Arcade Classics database.

Local data such as stats are stored locally on the user device using Chrome Storage Sync or localstorage if the users opts in.

Data Sharing

All in game statistics (high scores and games played) will be accessible to other users, accompanied by a random id and username.

Error reports can be optionally (opt in not opt out) shared with Arcade Classics.
